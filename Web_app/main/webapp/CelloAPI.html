/netsynth

/netsynth post

post

A Verilog logic specification is convered to wiring diagram (NOR/NOT gates) through logic synthesis. The result is not the final netlist; this endpoint serves to validate the Verilog code.

post /netsynth

A Verilog logic specification is convered to wiring diagram (NOR/NOT gates) through logic synthesis. The result is not the final netlist; this endpoint serves to validate the Verilog code.

Secured by basic
  • Request
  • Response

Query Parameters

  • verilog_text: required (string)

    File contents of a Verilog .v logic specification.

HTTP status code 200

Body

Type: text/plain

Example:

0  OUTPUT      (1)         
1  NOR         (3,2)       
2  NOT         (5)         
3  NOT         (4)         
4  INPUT                   
5  INPUT

HTTP status code 500

If the Verilog is not valid, errors within NetSynth result in a status 500 error.

Body

Type: application/json

Example:

{"timestamp":1450706258636,"status":500,"error":"Internal Server Error","exception":"java.lang.StringIndexOutOfBoundsException","message":"String index out of range: -1","path":"/netsynth"}

/submit

/submit post

post

Run Cello!

post /submit

Run Cello!

Secured by basic
  • Request
  • Response

Query Parameters

  • jobid: required (string)

    job name/identifier. A valid name starts with a letter, and only contains letters, numbers, and underscores. The regex for a valid name is [a-zA-Z][0-9a-zA-Z_]+

  • verilog: required (string)

    Verilog text (not filepath)

  • inputs: required (string)

    Input promoter data. Rows of space-separated name, low, high, dnaseq

    Example:

    pTac 0.0034 2.8 AACGATCGTTGGCTGTGTTGACAATTAATCATCGGCTCGTATAATGTGTGGAATTGTGAGCGCTCACAATT
  • outputs: required (string)

    Output gene data. Rows of space-separated name, dnaseq (each dnaseq should concatenate ribozyme, rbs, cds, terminator)

    Example:

    YFP CTGAAGCTGTCACCGGATGTGCTTTCCGGTCTGATGAGTCCGTGAGGACGAAAC
  • user_options: required (string)

    custom flags, not intended for new users. Space + dash separated strings.

    Example:

    -figures false -plasmid false -assignment_algorithm hill_climbing

HTTP status code 200

Body

Type: application/json

Example:

{
  "message": "SUCCESS: Loading results page."
}
